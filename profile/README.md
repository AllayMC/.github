## About Us

**AllayMC** is an open-source organization dedicated to developing high-quality server software for Minecraft: Bedrock Edition. Our mission is to provide a reliable, high-performance, and feature-rich server solution for the community.

### Our Vision

We believe that the Bedrock Edition community deserves server software that is:
- **Fast & Efficient** - Optimized for performance with minimal resource consumption
- **Developer-Friendly** - Clean APIs and comprehensive documentation for plugin development
- **Community-Driven** - Built with community feedback and contributions

### What We Do

- **[Allay](https://github.com/AllayMC/Allay)** - Our flagship server software, built from scratch with modern architecture
- **Ecosystem Tools** - Supporting libraries, plugins, and development tools
- **Documentation** - Comprehensive guides and API references at [docs.allaymc.org](https://docs.allaymc.org)

### Get Involved

We welcome contributions from developers of all skill levels! Whether you want to report bugs, suggest features, write documentation, or contribute code, there's a place for you in our community.

- Join our [Discord](https://discord.gg/ngkkE4hPTU) to chat with the team and community
- Check out our [GitHub repositories](https://github.com/AllayMC) to explore the code
- Read our [documentation](https://docs.allaymc.org) to get started
